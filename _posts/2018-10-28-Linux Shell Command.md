## install
CentOS: yum -y install nano  
Debian：apt-get install -y nano  

## line edit
Alt+6 : copy one line(复制一整行)  
Ctrl+K: cut one line(剪贴一整行)  
Ctrl+U: paste (粘贴)  
Ctrl+6: 标记，然后移动光标到 待复制／剪贴的文本末尾.若要取消，只需要再按一次Ctrl+6.  

## save
Ctrl+O: save (来保存所做的修改)  

## quit (退出)
Ctrl+X: quit
